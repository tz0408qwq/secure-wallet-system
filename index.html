P2P 交易引擎 & 託管

安全交易視窗、第三方託管、遊戲 API 驗證
買方資金鎖定於託管帳戶，賣方轉移物品後系統自動放款；糾紛時由平台介入仲裁。
  • 🔒 交易中顯示「託管保護」標籤
  • 📡 透過 Steam / 對應遊戲 API 確認物品所有權變更
  • 🤝 內建聲譽系統，降低詐騙風險

支付整合 & 先買後付 (BNPL)

法幣支付、規則型信用評估、彈性分期
整合信用卡、本地轉帳，並提供 BNPL 選項。信用模型使用平台歷史（帳戶年齡、交易次數、成功率）作為代理變數，短期還款（14天）。
  • 📊 BNPL 合規審查中，初期以內部規則引擎運行
  • ⚖️ 依用戶交易紀錄動態調整 BNPL 額度
  • ⏰ 還款提醒通知，逾期則限制帳戶功能

不可篡改交易總帳

稽核軌跡、完整金流日誌
所有存款、提款、託管、BNPL 還款、手續費均以區塊鏈式日誌儲存，提供監管單位及使用者完整查證。
  • 📜 每筆記錄含時間戳、雙方 ID、金額、交易狀態
  • 🔍 僅允許新增，無法刪改，滿足反洗錢要求
  • 📎 支援匯出報表 (CSV/PDF) 供稅務用途
整合安全與合規服務模組 — 即時行為分析、AI 詐欺偵測（基於市場分析報告中的信任缺口），自動標記可疑交易並強制 KYC 升級。
關鍵業務流程 (P2P 交易 & BNPL)
物品交易 + 託管流程
1. 賣方上架物品 (經遊戲API驗證所有權)
2. 買方點擊「立即購買」
3. 系統凍結買方資金至託管帳戶
4. 賣方於遊戲內發送物品，系統輪詢 API 確認交付
5. 確認無誤後，資金自託管釋放至賣方錢包
6. 交易完成，雙方互評，總帳記錄
BNPL (先買後付) 結帳流程
① 結帳時選擇「BNPL 先買後付」
② 規則引擎評估信用 (帳戶年齡>6個月、無違約紀錄、KYC Level≥2)
③ 審核通過，BNPL 額度扣款，錢包立即獲得對應金額完成交易
④ 產生還款計畫，推播通知提醒
⑤ 用戶以錢包餘額還款，全額清償後恢復信用額度

安全與便利平衡
